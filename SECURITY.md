# Security Policy

## Supported Versions

The following versions of Helix are currently supported with security updates:

| Version | Supported |
| :--- | :--- |
| 0.1.x | Yes (Development) |

## Reporting a Vulnerability

We take the security of Helix seriously. If you find a security vulnerability, please do NOT report it publicly via issue tracker. Instead, please report it directly to the maintainers.

### Reporting Process

1. Email the core development team at `security@helix-project.org` with details of the vulnerability.
2. Provide a detailed description, including steps to reproduce, potential impact, and any proof of concept code if available.
3. We will acknowledge receipt of your report within 48 hours and coordinate a fix.
4. We ask that you give us a reasonable amount of time to patch the issue before making it public.
