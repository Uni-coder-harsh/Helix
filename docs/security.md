# Security Architecture

Detailed documentation of safety guardrails, encryption methods, identity access management, and threat modeling.

> [!NOTE]
> Detailed threat modeling will be conducted in **Phase 2: Architecture** and hardened in **Phase 7: Production Hardening**.

## Security Guidelines

- Zero Trust network architecture.
- Encrypted variables and secrets using secure managers (e.g. GCP Secret Manager).
- Strict validation of external data inputs.
